Shared Component

An example of a component that can be shared between multiple other components (website pages).

  • A component is a discrete resource with its own URL.
  • You can visit this component at /components/static content.
  • The request is handled by a controller.
  • The controller can pass data to the component’s view.

  • Other components can call this component through their controllers.
