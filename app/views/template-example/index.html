This is the index view of template-example controller. It has been embedded in a template via its controller's "template" method. This method takes 2 arguments:

  1. The template name.
  2. The placeholder to supplant.

This view can be supplanted with model data in the usual way. In this example, the placeholder {example} is supplanted with the value 

{example}
.

Some data needs to be provided to the template into which this view is being embedded. For example, you might want to set the title element, or the h1 element. This can be also be done through the controller's view object. Just prefix any property that belongs to the template view with an underscore first.

For example, in order to supplant the template's <title> tag, add the following code to your controller action:

this.view._title = "My page title";

This concept works recursively. For example, if you are embedding this view's template into another template, you can use 2 underscores to reference its placeholders, and so on.
